# AfriCAN Manager - Gestion de la CAN 2025

## À propos
AfriCAN Manager est une application web conçue pour la gestion logistique de la Coupe d'Afrique des Nations 2025 au Maroc. Ce projet académique, réalisé à l'EMSI, relève un défi technique majeur : construire une architecture Single Page Application (SPA) complète en utilisant exclusivement du JavaScript Natif (Vanilla JS), sans dépendre d'aucun framework moderne (React, Angular, Vue, etc.).

L'objectif était de maîtriser les fondamentaux du web : manipulation du DOM, gestion de l'état, routage et persistance des données.

## Fonctionnalités
* **Tableau de Bord Interactif :** Visualisation des indicateurs clés (Stades, Équipes, Billets vendus) et graphiques dynamiques.
* **Gestion Complète (CRUD) :** Administration des stades, équipes et joueurs.
* **Billetterie :** Gestion des réservations et statuts des billets.
* **Expérience Utilisateur Avancée :** Mode Sombre (Dark Mode) natif et support multilingue complet (Français, Anglais, Arabe) avec gestion automatique du sens de lecture (RTL).
* **Persistance :** Sauvegarde des données via le LocalStorage pour simuler une base de données.

## Technologies
* **Langage :** JavaScript (ES6+ Vanilla)
* **Interface :** HTML5, CSS3
* **Bibliothèque Graphique :** Chart.js
* **Outils :** Visual Studio Code, Git
